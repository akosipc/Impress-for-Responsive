Responsive Web Design

By: Adrian Peterson Co
Bad Sector Software Labs

What in the world is Responsive Web Design or RWD?

A website that is created with the use of following stuff:

CSS3.0 Media Queries i.e "@media screen"

Fluid proportion-based grids i.e "width: 70%"

A little bit of imagination and everything that's nice!

A website that responds based on the environment

First coined by Ethan Marcotte in an A List Apart article.
Ranked #2 for Top Web Design Trends for 2012.
Nice kinks in doing Responsive Web Design
One layout fits all sizes and platforms
Future proof by using HTML5 and CSS3.0
Cross Platform Libraries are available
Lessens Time in Development

WHY?

“Day by day, the number of devices, platforms, and browsers that need to work with your site grows. Responsive web design represents a fundamental shift in how we’ll build websites for the decade to come.”
- Jeffrey Veen
