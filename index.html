Responsive Web Design

By: Adrian Peterson Co

What is Responsive Web Design?

It is a web design process that follows these steps:

CSS3.0 Media Queries

@media screen only

Fluid proportion-based grid

.columns {width: 70%}

Percentage Measurements

.fluid-font {font-size: 2em}

Meta Viewport in your HTML

meta name='viewport' content='width=device-width'

A website that responds based on the environment

First mentioned by Ethan Marcotte in his blog

A List Apart

Ranked #2 for Top Web Design Trends for 2012.
Nice kinks in doing Responsive Web Design
One layout fits all sizes and platforms
Future proof by using HTML5 and CSS3.0
Cross Platform Libraries are available
Numerous CSS Frameworks are now using it!
